Full portfolio website project

Includes:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: MongoDB

Pages:
- Home
- About Me
- Skills
- Projects
- Contact

How to run:
1. Extract the ZIP
2. Open terminal in the project folder
3. Run: npm install
4. Copy .env.example to .env
5. Run: npm run dev
6. Open: http://localhost:5000

Important:
- MongoDB must be installed and running for contact form storage
- If MongoDB is not running, website pages still open, but message saving will fail
